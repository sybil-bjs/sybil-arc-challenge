# ARC-AGI-3 Insights

Meta-learnings about how ARC-3 games work — patterns that apply across multiple games.

## Format

```yaml
id: ARC3-INS-XXX
timestamp: 2026-02-27
insight: "The meta-pattern or principle"
evidence: ["game1", "game2"] 
confidence: low|medium|high
applications: "How to use this insight"
tags: [exploration, mechanics, win-conditions]
```

## Categories

- **Exploration** — How to efficiently discover game mechanics
- **Mechanics** — Common patterns in how games work
- **Win Conditions** — Patterns in what triggers level completion
- **Efficiency** — How to minimize action count vs human baseline
- **Transfer** — Learnings that apply from one game to another
